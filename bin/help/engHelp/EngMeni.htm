Menu has following options:

  • File
    1. New - create new installer
    2. Open - open an existing installer
    3. Save - save current version of an installer
    4. Import - open multiple existing installers
    5. Exit - exit UIGUI
  • Edit
    1. Edit Node - edit name and values of installer, panel or parameter
    2. Cut - cut selected panel or parameter and move it to another location
    3. Copy - copy selected panel or parameter to another location
    4. Paste - place panel or parameter to selected location
    5. Move - move panel or parameter up or down in the list
  • Preview
    1. Properties - lists all parameters of an installer
  • Help
