Mnemonics make UIGUI easier to use. How do they work?
Example:
Pressing ALT+F will open File menu in the Menu bar.
Pressing ALT+F+N will after opening File menu, open a New File dialog.
List of all mnemonics:

  • F - opens File menu
  • F+N - New file
  • F+O - Open existing file
  • F+S - Save file
  • F+I - Import existing files

  • E - opens Edit menu
  • E+T - Edit
  • E+X - Cut
  • E+C - Copy
  • E+V - Paste
  • E+M - Move
  • E+M +U - Move up
  • E+M +D - Move down

  • W - opens View menu
  • W+P Properties

  • H - opens Help menu
  • H+L - Help
