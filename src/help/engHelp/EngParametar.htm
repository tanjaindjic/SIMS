Creating a parameter is possible by adding it's name and clicking on the "Add parameter" button.
Changing the name or value of selected parameter is possible by clicking on the button"Edit" in the Menu, followed by clicking on "Edit Node".
Removing selected parameter is done by clicking the "Remove" button.
Copying or cutting selected parameter to another panel is done by clicking on "Edit" in Menu, followed by "Copy", "Cut" or "Paste".
