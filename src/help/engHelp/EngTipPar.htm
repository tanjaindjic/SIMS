Following types of parameters can be added:

  1. Text - sentence or a text
  2. Text Line
  3. Picture
  4. Software location - location of the product to be installed
  5. Text File
  6. Company logo
  7. Default install location - predefined location where the product will be installed, changing it's value can be disabled
  8. Exe - executable command
  9. Install - adds Install button to the selected panel
  10. Checkbox - requires user's validation to proceed with installation
  11. Localization - adds the possibility of translating installer to another language. Only parameters added before localization will be translated. Symbol "=" is forbidden.
  12. List
